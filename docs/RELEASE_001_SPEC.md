# Release 001 — v1.0.0 MVP

**Название**: Kaizen MVP
**Цель**: Базовая система управления продуктами, задачами и релизами
**Дата создания**: 2026-02-28
**Дата выпуска**: 2026-02-28

---

## Цель релиза

Создать минимально жизнеспособный продукт (MVP) для отслеживания программных продуктов компании, сбора задач на улучшение и формирования релизов. Пользователь получает единый интерфейс для управления жизненным циклом улучшений.

---

## Scope (границы релиза)

### Включено (must)

- [x] CRUD продуктов (название, описание, репозиторий, стек, ответственный, путь к проекту)
- [x] Статусы продуктов (active / archived)
- [x] CRUD задач с типами (bug / improvement / feature) и приоритетами
- [x] Фильтрация задач по статусу
- [x] Создание релизов с выбором задач через чекбоксы
- [x] Публикация релиза (автоматическое обновление статусов задач)
- [x] Каскадное удаление (продукт → задачи + релизы)
- [x] Dark theme UI
- [x] REST API (13 эндпоинтов)

### Не входит (out of scope)

- Авторизация и RBAC
- Docker-деплой
- Git-интеграция (автотеги)
- Метрики и аналитика
- Уведомления
- Экспорт данных

## Архитектурные решения

### База данных

- Схема `opii`, префикс таблиц `kaizen_`
- Миграция: `001_initial_schema.sql`
- 4 таблицы: products, issues, releases, release_issues
- UUID первичные ключи, триггеры updated_at
- Транзакции в releases (create, update, delete, publish)

### Backend

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | /api/products | Список продуктов (с open_issues count) |
| POST | /api/products | Создать продукт |
| GET | /api/products/:id | Продукт по ID |
| PUT | /api/products/:id | Обновить продукт |
| DELETE | /api/products/:id | Удалить продукт (каскадно) |
| GET | /api/products/:id/issues | Задачи продукта (?status=) |
| POST | /api/issues | Создать задачу |
| GET | /api/issues/:id | Задача по ID |
| PUT | /api/issues/:id | Обновить задачу |
| DELETE | /api/issues/:id | Удалить задачу |
| GET | /api/products/:id/releases | Релизы продукта |
| POST | /api/releases | Создать релиз (с issue_ids[]) |
| GET | /api/releases/:id | Релиз с вложенными issues |
| PUT | /api/releases/:id | Обновить (add/remove issue_ids) |
| DELETE | /api/releases/:id | Удалить (issues → open) |
| POST | /api/releases/:id/publish | Опубликовать релиз |

### Frontend

| Страница | Файлы | Описание |
|----------|-------|----------|
| Продукты | index.html + products.js | Сетка карточек, модал создания/редактирования |
| Продукт | product.html + product.js | Шапка, таблица задач, карточки релизов, 3 модала |
