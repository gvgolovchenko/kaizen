## 3. Технический долг

> Оценка: **8/10 (ХОРОШО)**
> Нет TODO/FIXME/HACK/XXX, нет захардкоженных секретов, console.log полностью мигрирован на pino, есть тесты (3 файла, все проходят).

### Высокий приоритет

**1. process-runner.js — 2470 строк**
Самый большой файл проекта. Содержит логику для всех типов процессов + smoke testing + checkpoint tracking + деплой.
Рекомендация: разбить на `develop-phase.js`, `deploy-phase.js`, `test-runner.js`.

**2. routes/api.js — 2041 строка**
Все API-эндпоинты в одном файле (70+ обработчиков).
Рекомендация: разбить по доменам: `routes/products.js`, `routes/releases.js`, `routes/scenarios.js`, `routes/rc.js`, `routes/gitlab.js`.

**3. GitLab: токен в URL.password** (`server/gitlab-client.js:26`)
`url.password = gl.access_token` — `url.toString()` включает токен в строку, может утечь в логи.
Рекомендация: строить auth URL напрямую строкой.

### Средний приоритет

**4. Silent `.catch(() => {})` — 39 шт.**

| Файл | Количество |
|------|------------|
| process-runner.js | 21 |
| ai-caller.js | 6 |
| routes/api.js | 5 |
| scenario-runner.js | 3 |
| scheduler.js | 2 |
| gitlab-client.js | 1 |
| smoke-tester.js | 1 |

Рекомендация: для уведомлений заменить на `.catch(err => log.warn(...))`.

**5. Дублирование git remote setup** (`server/gitlab-client.js`)
`pushToGitlab()` и `pushToDefaultBranch()` содержат одинаковый код.
Рекомендация: извлечь в `ensureGitRemote(projectPath, remoteName, authUrl)`.

**6. Рассинхрон версии в package.json**
`package.json` — `"version": "1.11.0"`, реально 1.19.0.
Рекомендация: обновлять version при публикации релиза.

### Низкий приоритет

**7.** Нет линтера и pre-commit хуков (ESLint, Prettier, husky).

**8.** Тесты — только фронтенд (3 файла). Нет unit/integration тестов для server/.

**9.** `playwright` в `dependencies` вместо `optionalDependencies` (~300 МБ с браузерами).

### Что чисто

- Нет TODO/FIXME/HACK/XXX в коде
- Нет console.log/error/warn (полная миграция на pino)
- Секреты только из .env, API-ключи маскируются в ответах
- Нет `|| true` в скриптах/Dockerfile
- Нет закомментированных блоков кода
- Минимальное дублирование (<10 строк)
